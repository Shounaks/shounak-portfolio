## Adding Skills to opencode

Skills extend opencode's capabilities. Browse available skills at [mcpservers.org/agent-skills](https://mcpservers.org/agent-skills).

### Planning

```bash
npx skills add https://github.com/mattpocock/skills --skill grill-me
npx skills add https://github.com/mattpocock/skills --skill grill-with-docs
npx skills add https://github.com/mattpocock/skills --skill to-prd
npx skills add https://github.com/mattpocock/skills --skill review
```

### Agentic Development

npx skills add https://github.com/vercel-labs/skills --skill find-skills

## MCP Server Configuration

Add these MCP servers to your `opencode.json`:

```json
{
  "mcpServers": {
    "next-devtools-mcp": {
      "type": "local",
      "command": ["npx", "-y", "next-devtools-mcp@latest"],
      "enabled": true
    },
    "tailwind-mcp": {
      "type": "local",
      "command": ["npx", "-y", "tailwindcss-mcp-server"],
      "enabled": true
    },
    "shadcn": {
      "type": "local",
      "command": ["npx", "shadcn@latest", "mcp"],
      "enabled": true
    }
  },
  "lsp": true
}
```
